### **To Do List란?**

-   나중에 할 일을 까먹지 않게 간단하게 기록할 수 있는 프로그램입니다.
-   할 일을 추가, 삭제 또는 완료로 표시할 수 있습니다.
-   프로그램 종료 시 기록된 할 일들은 데이터 파일로 저장되어 프로그램을 다시 시작해도 그대로 볼 수 있습니다.
